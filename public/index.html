API Route Documentation

GET /spotlight

Returns spotlight/featured content.

GET /spotlight

GET /suggestions

Returns search suggestions and featured content.

GET /suggestions

GET /search

Search for movies, series, and other content with advanced filtering.

Parameter Type Description Required? Default
keyword string The search query (e.g., title) Yes —
page number Page number for pagination No 1
genre string Filter by genre No —
year number Filter by release year No —
country string Filter by country No —
GET /search?keyword=avengers&genre=action&year=2023&country=usa&page=1

GET /genres

Returns a list of all available genres with their content counts.

GET /genres

GET /genre/:genre

Returns content for a specific genre with pagination.

Parameter Type Description Required? Default
genre string Genre slug (URL parameter) Yes —
page number Page number for pagination No 1
GET /genre/action?page=1

GET /category/:category

Returns content for a specific category with optional filtering.

Parameter Type Description Required? Default
category string Category slug. Possible values: recommended, episode, movies, series, top movies, most viewed, director, actors, studios, countries, networks, genres, release-year Yes (URL param) —
page number Page number for pagination No 1
filter string Filter by subcategory (e.g., movies, series, genre, most-viwed) No —
country string Filter by country code (e.g., France, Japan, Philippines, Korea) No —
GET /category/movies?page=1&country=usa

GET /seasons/:type/:id

Returns seasons information for a series.

Parameter Type Description Required? Default
type string Type of media (series or episode) Yes (URL param) —
id string ID of the media item Yes (URL param) —
GET /seasons/series/12345
